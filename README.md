## Installation & Usage
Clone the repo.  
Run commands:
```shell
$ cd <project folder>
$ make install
```
Choose one of the games listed below:
```shell
$ brain-even
$ brain-calc
$ brain-gcd
$ brain-progression
$ brain-prime
```
